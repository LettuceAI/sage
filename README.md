# SAGE

**S**afety **A**nd **G**uardrails **E**ngine — lightweight multi-label content moderation model for chat and roleplay applications.

Built for [LettuceAI](https://github.com/LettuceAI). Runs locally, no API calls, no data leaves the device.

## Features

- **7 categories:** NSFW, violence, harassment, hate speech, self-harm, grooming, sexual content involving minors
- **Trajectory-aware:** scores a message given its conversation history, not in isolation — catches grooming escalation, coerced consent, and ideation build-up that message-level classifiers miss
- **~35MB INT8** model size
- **~30–50ms CPU inference** on mid-range hardware at full 1024-token input (faster for shorter)
- **ONNX runtime** — works in Python and Rust
- Multi-label output with per-category confidence scores
- Configurable per-category thresholds

## Architecture

Fine-tuned [`jinaai/jina-embeddings-v2-base-en`](https://huggingface.co/jinaai/jina-embeddings-v2-base-en) (137M params, 12 layers, dim 768, ALiBi, native 8192 context) with role-token embeddings and a linear classifier head. Max input 16 turns / 1024 tokens. See [`docs/architecture.md`](docs/architecture.md) for details.

## Quick start

> Not yet released. Early development — targeting v1 in Q2 2026.

### Python

```python
from sage import Sage

sage = Sage.from_pretrained("lettuceai/sage-v1")

# Single message
result = sage.check("your text here")

# Conversation-aware (recommended — catches trajectory-level patterns)
result = sage.check([
    {"role": "user", "text": "earlier message"},
    {"role": "char", "text": "reply"},
    {"role": "user", "text": "message to classify"},  # last turn is scored
])
# {"flagged": True, "categories": {"grooming": {"score": 0.71, "flagged": True}, ...}}
```

### Rust

```rust
use sage::Sage;

let sage = Sage::load("sage-v1-int8.onnx")?;
let result = sage.check("your text here")?;
```

## Training data

SAGE is trained on an aggregated, deduplicated, hard-negative-mined dataset spanning six commercially-permissive sources. See [`DATA_LICENSES.md`](DATA_LICENSES.md) for full attribution.

Rare categories (grooming, self-harm) are augmented with synthetic examples generated by larger LLMs and human-reviewed.
